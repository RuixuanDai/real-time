## Prerequistes for UI
  - Node,js
  - Other dependences run the following
     ```
     $ npm install jquery semantic-ui-css handlebars vanilla-router express dotenv axios
     ```
   
## Starting App
  - Navigate to the following path
   ```
   ~/real-time/web-app
   $ node server.js
   ```
  - App will run on http://localhost:8080/
